# Gabinet Streamlit

Aplikacja gabinetu lekarskiego napisana w Streamlit.

## Uruchomienie lokalne (pakiet)

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .

gabinet-streamlit
```

Alternatywnie bez instalacji pakietu:

```bash
pip install -r requirements.txt
streamlit run app.py
```
